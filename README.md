# Raid and Run

You are trapped in a deadly dungeon. Bad news.
There are shiny coins everywhere. Good news!
Giant fireballs are screaming down the corridor. Very bad news. Move fast!

A pixel-art browser game built with Vite, TypeScript, and Canvas.

## Play

https://codingcodey.github.io/raid-and-run/

## Local Development

```sh
npm install
npm run dev
```

## Checks

```sh
npm test
npm run build
```

## Deployment

This project is configured for GitHub Pages through GitHub Actions. Push to the `main` branch, then set the repository's Pages source to GitHub Actions in GitHub's repository settings.
